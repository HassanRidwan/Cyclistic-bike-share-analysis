# Cyclistic Bike-Share Analysis

This project analyzes Cyclistic bike-share usage to compare behavior between
member and casual riders.

## Objective
Identify usage patterns and insights that can help convert casual riders
into annual members.

## Key Insights
- Members take more rides overall but for shorter durations.
- Casual riders take fewer rides but ride longer, especially on weekends.
- Weekend and leisure-focused incentives present an opportunity for conversion.
